# AI Prompt Library

Welcome to the AI Prompt Library ‚Äì a comprehensive, self-hosted repository of AI prompts, coding standards, and documentation templates.

## What is This?

The AI Prompt Library is your centralized source of truth for:

- **AI Prompts** - Reusable prompts for Claude, ChatGPT, and other AI models
- **Coding Standards** - Organization-wide standards and best practices
- **Agent Prompts** - Complete autonomous agents for specific tasks
- **Feature Guides** - How-to documentation and implementation guides
- **Security Rules** - Security standards and compliance checklists

All prompts are version-controlled, searchable, and exportable to your preferred platforms.

## Export Commands

Export prompts in bulk to your preferred format:

```bash
# Export all formats
npm run export:all

# Export specific format
npm run export:claude     # Claude JSON
npm run export:chatgpt    # ChatGPT format
npm run export:raycast    # Raycast commands
npm run export:csv        # Spreadsheet format
npm run export:markdown   # Markdown bundle
```

Exports are saved to `data/exports/` with timestamps.

## Best Practices

### Before Using a Prompt

- Always test with your AI model first
- Customize for your specific context
- Review examples in the prompt
- Check related prompts for context

### When Creating a Prompt

- Be specific and clear in instructions
- Provide concrete examples
- Include expected output format
- Add relevant tags for discoverability
- Document any special requirements
